# Características de Markdown

Docusaurus soporta **[Markdown](https://daringfireball.net/projects/markdown/syntax)** y algunas **características adicionales**.

## Páginas preliminares

Los documentos de markdown tienen metadatos en la parte superior llamados [Front Matter](https://jekyllrb.com/docs/front-matter/):

```text title="my-doc.md"
// highlight-start
---
id: my-doc-id
title: My document title
description: My document description
slug: /my-custom-url
---
// highlight-end

## Encabezado de Markdown

Texto con Markdown [links](./hello.md)
```

## Enlaces

Se admiten enlaces regulares de Markdown, utilizando rutas url o rutas de archivos relativas.

```md
Veamos cómo [Crear una página](/create-a-page).
```

```md
Veamos cómo [Crear una página](./create-a-page.md).
```

**Resultado:** Veamos cómo [Crear una página](./create-a-page.md).

## Imágenes

Se admiten imágenes de Markdown regulares.

Puede utilizar rutas absolutas para hacer referencia a imágenes en el directorio estático (static/img/docusaurus.png):

```md
![Docusaurus logo](/img/docusaurus.png)
```

![Docusaurus logo](/img/docusaurus.png)

También puede referenciar imágenes relativas al archivo actual. Esto es particularmente útil para colocar imágenes cerca de los archivos Markdown que las utilizan:

```md
![Docusaurus logo](./img/docusaurus.png)
```

## Bloques de código

Los bloques de código Markdown son compatibles con el resaltado de sintaxis.

````md
```jsx title="src/components/HelloDocusaurus.js"
function HelloDocusaurus() {
  return <h1>Hello, Docusaurus!</h1>;
}
```
````

```jsx title="src/components/HelloDocusaurus.js"
function HelloDocusaurus() {
  return <h1>Hello, Docusaurus!</h1>;
}
```

## Advertencias

Docusaurus tiene una sintaxis especial para crear advertencias y callouts:

```md
:::tip Mi TIP

Utiliza esta impresionante característica

:::

:::danger Tenga cuidado

Esta acción es peligrosa

:::
```

:::tip Mi tip

Utiliza esta impresionante característica

:::

:::danger Ten cuidado

Esta acción es peligrosa

:::

## MDX y Componentes React

[MDX](https://mdxjs.com/) puede hacer tu documentación más **interactiva** y permite el uso de cualquier componente **React dentro de Markdown**:

```jsx
export const Highlight = ({children, color}) => (
  <span
    style={{
      backgroundColor: color,
      borderRadius: '20px',
      color: '#fff',
      padding: '10px',
      cursor: 'pointer',
    }}
    onClick={() => {
      alert(`You clicked the color ${color} with label ${children}`)
    }}>
    {children}
  </span>
);

Esto es <Highlight color="#25c2a0">Docusaurus verde</Highlight> !

Esto es <Highlight color="#1877F2">Facebook azul</Highlight> !
```
