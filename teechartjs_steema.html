Elevate Your Data with TeeChart

Javascript charting library plotting graphs to all compatible browsers in native Javascript format using the HTML5 Canvas element.

Get started any way you want

Dive straight into development with TeeChart. You can opt for the CDN, install it using a package manager, or simply download the source code to get started.

Packet manager

You can set up TeeChart's source Sass and Javascript files via npm, RubyGems, Composer, or Meteor for convenient package management. It's worth noting that package-managed installations won't provide documentation or our complete build scripts. If you're looking for a rapid way to kickstart your Bootstrap projects, you can also make use of any of the demos available in our Examples repository.

CDN

If you only requiere TeeChart's compiled CSS or JS, you have the option to leverage jsDelivr. Witness it in action by exploring our straightforward quick start guide, or peruse the examples to give your next project a flying start. Additionally, you have the flexibility to include Popper and our Javascript components separately as needed.

Example

TeeChart is a Javascript charting library that plots graphs to all compatible browsers in native Javascript format using the HTML5 Canvas element. TeeChart is an interactive Javascript chart for visualization of data. Designed for web applications, TeeChart offers a great user experience and a large number of chart types for visualization of data, including bar, line, area, pie, bubble, scatter, polar, and financial charts. This browser does not seem to support HTML5 Canvas.
